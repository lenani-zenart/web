CSS folosit pentru a seta imaginile de fundal mari cu anumite proprietati

Imagini mari de fundal

background-image: url(); - sursa imaginii

background-size: cover; - afiseaza imaginea de fundal acoperind tot spatiul elementului HTML

background-repeat: no-repeat; - nu permite imaginii sa se repete

background-position: center center; - seteaza imaginea pe centrul orizontal si centrul vertical in spatiul elementului HTML

background-attachment: fixed; - seteaza priprietatea imaginii de fundal sa ramana intr-un punct fix fata de pozitia elementului HTML in pagina

CSS folosit pentru a seta imaginile de fundal mici cu anumite proprietati

Imagini mici setate ca fundal

background-image: url(); - sursa imaginii

background-size: auto; - afiseaza imaginea de fundal folosind dimensiunea reala a imaginii

background-repeat: repeat-x; - permite imaginii sa se repete pe orizontala

background-position: 0 0; - seteaza imaginea in interiorul elementului HTML afisando de la stanga sus (primul 0) si de sus in jos (al doilea 0) / left top
