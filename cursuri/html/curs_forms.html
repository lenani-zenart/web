Initial

<form> este elementul eticheta HTML care defineste un formular HTML pentru intrarea (type/input) utilizatorilor

<form action="/pagina_contact.php">
Nume:
<input type="text" name="nume" value="Lenani">
Prenume:
<input type="text" name="prenume" value="Art">

Mesaj:
<textarea name="mesaj" type="text"></textarea>

<input type="submit" value="Trimite"> </form>

Action / Atributul acțiune

Atributul de acțiune definește acțiunea care trebuie efectuată la trimiterea formularului.
În mod normal, datele formularului sunt trimise către o pagină web de pe server când utilizatorul face clic pe butonul Submit/Trimiteți.
În exemplul de mai sus, datele formularului sunt trimise către o pagină de pe server numită "/pagina_contact.php". Această pagină conține un script de pe server care gestionează datele formularului:

<form action="/pagina_contact.php">

Elementul <input>

Cel mai important element a unui formular

<input type="text"> definește un câmp de introducere dintr-o singură linie pentru introducerea textului

<input type="radio"> definește butoanele radio (select)

Butoanele radio (select) permite selectarea unei singure optiuni dintr-o lista predefinita

Rosu
Albastru
Galben

<form>
<input type="radio" name="culoare" value="rosu" checked> Rosu<br>
<input type="radio" name="culoare" value="albastru"> Albastru<br>
<input type="radio" name="culoare" value="galben"> Galben
</form>

Atributul method

Acest formular va fi trimis folosind metoda GET:

<form action="/pagina_contact.php" method="get"> sau <form action="/pagina_contact.php" method="post">

Atributul metode specifică metoda HTTP (GET sau POST) care trebuie utilizată la trimiterea datelor de formular:

Metoda implicită la trimiterea datelor din formular este GET.
Cu toate acestea, când se utilizează GET, datele formularului trimis vor fi vizibile în câmpul adresei paginii:

Nume:

Prenume:

Mesaj:


Atributul name / nume

Fiecare câmp de intrare trebuie să aibă un atribut nume care trebuie trimis.
Dacă atributul name este omis, datele acelui câmp de intrare nu vor fi trimise deloc.
Acest exemplu va trimite numai câmpul de introducere "Nume":

Nume:

Prenume:
