Lista taguri HTML

Tag Descriere
<!--...--> Defineste un comentariu
<!DOCTYPE>  Defineste tipul documentului
<a> Defineste un hyperlink (un link)
<abbr> Defineste o abreviere sau un acronim
<acronym> Nu este suportat in HTML5. Foloseste <abbr>  . - Defineste un acronym
<address> Defineste informațiile de contact ale autorului / proprietarului unui document
<applet> Nu este suportat in HTML5. Foloseste <embed> sau <object>  . - Defineste un applet încorporat
<area> Defineste o zonă din interiorul unei imagini
<article> Defineste un articol
<aside> Defineste conținutul în afară de conținutul paginii
<audio> Defineste continut audio
<b> Defineste bold text
<base> Specifică adresa URL de bază / țintă pentru toate adresele URL relative într-un document
<basefont> Nu este suportat in HTML5. Foloseste CSS. Specifică o culoare, o dimensiune și un font prestabilit pentru toate textele dintr-un document
<bdi> Izolează o parte a textului care ar putea fi formatat într-o direcție diferită de alt text în afara acestuia
<bdo> Impune o alta directie de redare text fata de direcția curentă de text
<big> Nu este suportat in HTML5. Foloseste CSS . - Defineste text mare
<blockquote> Defineste o secțiune citată dintr-o altă sursă
<body> Defineste corpul documentului
<br> Defineste o pauză de o singură linie
<button> Defineste un buton clicabil
<canvas> Folosit pentru a desena elemente grafice folosind program scipt (limbaj de programare ca JavaScript)
<caption> Defineste legenda
<center> Nu este suportat in HTML5. Use CSS . - Defineste text centrat
<cite> Defineste titlu citat
<code> Defineste formatarea text afisand codul sursa
<col> Specifică proprietățile coloanelor pentru fiecare coloană dintr-un element <colgroup>  
<colgroup> Specifică un grup de una sau mai multe coloane. Mai multe coloane într-un tabel
<data> Reda conținutul dat cu o traducere care poate fi citită de mașină
<datalist> Specifică o listă de opțiuni predefinite pentru comenzile de intrare
<dd> Defineste o descriere / valoare a unui termen dintr-o listă de descriere
<del> Defineste textul care a fost șters dintr-un document
<details> Defineste detalii suplimentare pe care utilizatorul le poate vizualiza sau ascunde
<dfn> Reprezintă instanța definitorie a unui termen
<dialog> Defineste o casetă de dialog sau o fereastră
<dir> Nu este suportat in HTML5. Foloseste <ul>  . - Defineste o listă de directoare
<div> Defineste o sectiune in pagina
<dl> Defineste o descriere într-o listă
<dt> Defineste un termen / nume într-o listă
<em> Defineste text subliniat 
<embed> Defineste un container pentru o aplicație externă (non-HTML)
<fieldset> Gruparea elementelor într-o formă
<figcaption> Defineste o legendă pentru un element <figure>
<figure> Specifică conținutul autonom
<font> Nu este suportat in HTML5. Use CSS . - Defineste font, culoare, si dimensiune font (text)
<footer> Defineste sectiunea de subsol a paginii web
<form> Definesteun formular HTML pentru intrarea (type/input) utilizatorilor
<frame> Nu este suportat in HTML5. - Defineste o fereastră (un cadru) într-un set de cadre
<frameset> Nu este suportat in HTML5. - Defineste set de cadre
<h1> to <h6> Defineste rubrici HTML
<head> Defineste informații despre document
<header> Defineste un antet pentru un document sau o secțiune
<hr> Defineste o schimbare tematică a conținutului
<html> Defineste rădăcina unui document HTML
<i> Defineste o parte din text redat ca voce sau stare de spirit alternativă
<iframe> Defineste un cadru inline
<img> Defineste o imagine
<input> Defineste control de intrare (type, input)
<ins> Defineste un text care a fost inserat într-un document
<kbd> Defineste intrare tastatură
<label> Defineste o etichetă pentru un element <input>
<legend> Defineste o legendă pentru un elemnt <fieldset>
<li> Defineste un element dintr-o listă
<link> Defineste relația dintre un document și o resursă externă (cea mai utilizată pentru a conecta la foile de stil)
<main> Specifică conținutul principal al unui document
<map> Defineste image-map client-side
<mark> Defineste text marcat / evidențiat
<meta> Defineste metadate despre un document HTML
<meter> Defineste o măsurare scalară într-un interval cunoscut
<nav> Defineste link-uri de navigare
<noframes> Nu este suportat in HTML5. - Defineste un conținut alternativ pentru utilizatorii care nu acceptă cadre
<noscript> Defineste un conținut alternativ pentru utilizatorii care nu acceptă scripturile de pe partea clientului
<object> Defineste un obiect încorporat
<ol> Defineste o listă ordonată
<optgroup> Defineste un grup de opțiuni asociate într-o listă derulantă
<option> Defineste o opțiune dintr-o listă drop-down
<output> Defineste rezultatul vizibil redat
<p> Defineste un paragraf
<param> Defineste un parametru pentru un obiect
<picture> Defineste un container pentru multiple imagini
<pre> Defineste text 
preformatat
<progress> Represents progresul unei sarcini
<q> Defineste o cotație scurtă
<rp> Defineste ce să afișați în browsere care nu acceptă adnotări ruby
<rt> Defineste o explicație / pronunțare a caracterelor (pentru tipografia din Asia de Est)
<ruby> Defineste o adnotare ruby(pentru tipografia din Asia de Est)
<s> Defineste textul care nu mai este corect
<script> Defineste un script
<section> Defineste o sectiune intr-un document
<select> Defineste o listă drop-down
<small> Defineste text micsorat
<source> Defineste mai multe resurse media pentru elementele media (<video> si <audio>)
<span> Defineste o sectiunea intr-un element HTML
<strike> Nu este suportat in HTML5. Foloseste <del> sau <s>  . - Defineste text strikethrough
<strong> Defineste text important
<style> Defineste informații despre stil (CSS) pentru un document
<sub> Defineste textul înscris sub
<summary> Defineste un element sumar vizibil <details>
<sup> Defineste text suprascript sup
<svg> Defineste un container pentru grafica SVG
<table> Defineste un tabel
<tbody> Grupează conținutul corpului într-un tabel
<td> Defineste o celula intr-un tabel
<template> Defineste un șablon
<textarea> Defineste (zona de text) care permite mai multe linii
<tfoot> Grupează conținutul subsolului într-un tabel
<th> Defineste o celulă de antet într-o tabelă
<thead> Grupează conținutul antetului într-un tabel
<time> Defineste data/ora
<title> Defineste un titlu pentru document
<tr> Defineste un rând dintr-un tabel
<track> Defineste text pentru elementele media (<video> si <audio>)
<tt> Nu este suportat in HTML5. Use CSS . - Defineste teletype text
<u> Defineste text care ar trebui să fie diferit fata de textul obișnuit
<ul> Defineste o listă neordonată
<var> Defineste o variabila
<video> Defineste un videoclip sau un film
<wbr> Defineste o posibilă întrerupere a liniei
