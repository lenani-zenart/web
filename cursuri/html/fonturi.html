Fonturile sistem - Fonturile personalizate - Fonturile Web

Fonturile joaca un rol important in pagina web, atat din punct de vedere vizual pentru ca ne permite sa diferentiem anumite elemente sau sectiuni dar, atentie, si din punct de vedere "tehnic".

Diferenta dintre fonturile sistem si fonturile web este aceea ca fonturile sistem sunt cele instalate in pc-ul / laptopul tau si se regasesc in toate aceste dispozitive.

Fonturile personalizate, sunt fonturile pe care le descarci de pe net si le instalezi in pc/laptop.

Web Fonts sau fonturile web sunt fonturile care NU trebuie sa le descarci (sau sa le instalezi) in pc/laptop dar care sunt totusi redate de browser deoarece sunt gazduite online

Exemple de fonturi sistem

Fonturi sistem pe care le ai instalate in dispozitivul tau (default font system). In sistemul de operare Windows poti gasi aceste fonturi in partitia C:\Windows \Fonts

In cele mai multe cazuri vei observa ca in aceasi sintaxa css ai sa vezi mai multe denumiri de fonturi despartite prin virgula. Primul font este cel care trebuie redat, in cazul in care vizitatorul paginii web NU are acel font instalat, dispozitivul va afisa al doilea font, sau al treilea font (font fallback).
Pentru a "forta" redarea primului font, se folosesc gilimelele "Verdana", Geneva, sans-serif sau 'Verdana', Geneva, sans-serif
Daca mut ghilimelele pentru fontul Geneva, dar fara sa ii schimb pozitia in ordine, fortez ca acest font sa aiba prioritate
Verdana, 'Geneva', sans-serif - chiar daca seria de fonturi incepe cu Verdana, fortez ca textul afisat sa foloseasca fontul Geneva cu inlocuitor sans-serif.

Fiecare dispozitiv vine cu propria selectie de fonturi preinstalata. Selectia se bazeaza in mare parte pe sistemul sau de operare.
Problema este ca fiecare sistem difera putin.
Dispozitivele bazate pe Windows pot avea un singur grup. Cele MacOS folosesc alta. Sistemul Android utilizeaza si el propriul sistem de fonturi compatibile cu Android.
Exemplu: Sa presupunem ca designerul a ales o familie de fonturi obscure si platite pentru designul acestei pagini. Daca nu aveti fontul deja instalat si nu se ofera versiunea host - fontul pe care îl vedeti ar reveni la unele variante de bază (fallback), cum ar fi Verdana sau sans-serif (in acest caz).

font sistem
font-family: Verdana, Geneva, sans-serif
font sistem
font-family: Georgia, 'Times New Roman', Times, serif
font sistem
font-family: 'Lucida Console', Monaco, monospace

Aceste fonturi nu au nevoie de host, safe for web

Exemple de fonturi web

Fonturile se pot clasifica in mai multe categorii deoarece, fiecare font are anumite proprietati de care ne putem folosi

Ca sa nu sar direct in exemple, mai intai doresc sa va arat 3 metode de a folosi fonturile gazduite in pagina web

Vom folosi serviciul gratuit Google Fonts

Pentru a putea folosi fonturi web in pagina, aceste fonturi trebuie sa fie gazduite online (incarcate online). Fonturile web NU folosesc acceasi extensia ca cele sistem

Prima metoda de a adauga un anumit font pentru a fi folosit in pagina web este prin adaugarea link-ului intre etichetele <head> si </head>
<!DOCTYPE html>
<html lang="ro-RO">
<head>
<meta charset="UTF-8">
<title>Web Fonts</title>
<link href="https://fonts.googleapis.com/css?family=Lobster&display=swap" rel="stylesheet">

...
</head>

Metoda a doua este prin import. In acest caz, fontul va fi adaugat in foaia de stil (css)


<style>
@import url('https://fonts.googleapis.com/css?family=Lobster&display=swap');
</style>

In pasi precedenti am adaugat un font ca resursa pentru pagina web, acum urmeaza sa folosim acest font in elementele text.
Pentru ca un paragraf de exemplu sa poata utiliza acest font, sa redea din punct de vedere vizual atributele acestui font, va trebui sa specificam in CSS acest lucru.
Sintaxa de baza

font-family: 'Lobster', cursive;

Cod CSS

h2{font-family: 'Lobster', cursive;}

Va afisa:

Acest element foloseste fontul Lobster

Putem folosi mai multe fonturi in pagina, librarii de fonturi dar nu este recomandat sa folosim mai mult de 3 fonturi deoarece va avea un impact negativ asupra incarcarii paginii. Unele fonturi consuma resurse host mai mult de cat altele
