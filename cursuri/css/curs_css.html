CSS

CSS este marcajul care descrie stilul unui document HTML.
CSS descrie modul în care ar trebui afișate elementele HTML.
Acest material vă va învăța CSS de bază și celele mai populare proprietăți

Proprietati Descriere

A

  
animation Proprietate CSS atribuita pentru toate derivatele de animație
animation-*
animation-delay  Specifică o întârziere pentru începerea unei animații
animation-delay: 2s;
animation-direction Specifică dacă o animație trebuie redată înainte, înapoi sau în cicluri alternative
animation-direction: alternate;
animation-duration Specifică cât timp va dura o animație pentru a finaliza un ciclu
animation-duration: 3s;
animation-fill-mode Specifică un stil pentru element atunci când animația nu se redă (înainte de a începe, după ce se termină sau ambele)
animation-fill-mode: forwards;
animation-iteration-count Specifică numărul de repetări ale unei animații
animation-iteration-count: 2;
animation-name Specifică un nume pentru animația @keyframes
animation-name: shimbaculoare;
animation-play-state Specifică dacă animația rulează sau este întreruptă
animation-play-state: paused;
animation-timing-function Specifică curba de viteză a unei animații
animation-timing-function: linear;

B

  
background Proprietate globala atribuita pentru toate derivatele de fundal
background-*
background-attachment Setează dacă o imagine de fundal se derulează cu restul paginii sau este fixată
background-image: url("logo.gif"); background-repeat: no-repeat; background-attachment: fixed;
background-blend-mode Specifică modul de redare pentru fiecare layer de fundal (culoare / imagine)
background-blend-mode: normal|multiply|screen|overlay|darken|lighten|color-dodge|saturation|color|luminosity;
background-clip Definește cât de mult ar trebui să se extindă fundalul (culoarea sau imaginea) într-un element
background-clip: border-box|padding-box|content-box|initial|inherit;
background-color Specifică culoarea de fundal a unui element
background-color: color|transparent|initial|inherit;
background-image Specifică una sau mai multe imagini de fundal pentru un element
background-image: url|none|initial|inherit;
background-origin Specifică poziția de origine a unei imagini de fundal
background-origin: padding-box|border-box|content-box|initial|inherit;
background-position Specifică poziția unei imagini de fundal
background-position: top center;
background-repeat Setează dacă / cum va fi repetată o imagine de fundal
background-repeat: repeat|repeat-x|repeat-y|no-repeat|initial|inherit;
background-size Specifică dimensiunea imaginilor de fundal
background-size: auto|100%|cover|contain|initial|inherit;
border Proprietate pentru lățimea bordurii, stilul bordura și culoarea
border: 2px solid #000;
border-radius Proptietate ce rotunjeste colturile unui element -*-border-radius
border-radius: 8px;
bottom Setează poziția elementelor in partea inferioară a elementului părinte
bottom: auto|20px|initial|inherit;
box-decoration-break Setează comportamentul fundalului și marginii unui element la întreruperea paginii sau, în cazul elementelor în linie, la ruperea liniei.
box-decoration-break: slice|clone|initial|inherit|unset;
box-shadow Adaugă una sau mai multe umbre la un element
box-shadow: 5px 10px #888888;
box-sizing Definește modul în care se calculează lățimea și înălțimea unui element: ar trebui să includă padding și margins sau nu
box-sizing: content-box|border-box|initial|inherit;

C

  
caption-side Specifică plasarea unei legende de tabel
caption-side: top|bottom|initial|inherit;
caret-color Specifică culoarea cursorului (caret) în intrări, textare sau orice element care poate fi editabil
caret-color: auto|color;
@charset Specifică codificarea caracterelor utilizate în foaia de stil
@charset "UTF-8";
clear Specifică pe care laturi ale unui element elementele plutitoare nu au voie să plutească
clear: none|left|right|both|initial|inherit;
clip Decupeaza/taie un element absolut pozitionat
clip: auto|shape|initial|inherit;
color Specifică culoarea textului
color: #000;
column-count Specifică numărul de coloane in care un element trebuie împărțit
column-count: 4|auto|initial|inherit;
column-fill Specifică modul de umplere a coloanelor, echilibrat sau nu
column-fill: balance|auto|initial|inherit;
column-gap Specifică distanta dintre coloane
column-gap: 22px|normal|initial|inherit;
column-rule Proprietate globala pentru column-rule-*
column-rule: 4px double #ff00ff;
content Folosit cu :before și :after pseudo-elemente, pentru a insera conținutul generat
content: normal|none|counter|attr|string|open-quote|close-quote|no-open-quote|no-close-quote|url|initial|inherit;
cursor Specifică cursorul mouse-ului care va fi afișat când interactioneaza cu un element specific
cursor: pointer;

D

  
direction Defineste directia de scriere (ltr = left to right/ dela stanga la dreapta)
direction: ltr|rtl|initial|inherit;
display Specifică modul în care trebuie afișat un anumit element HTML
display: none|inline|block|inline-block;

E

  
empty-cells Specifică dacă doriți sau nu să afișați bordura sau culoarea de fundal intr-un tabel
empty-cells: show|hide|initial|inherit; 

F

  
filter Definește efectele (de exemplu, estomparea sau schimbarea culorii) pe un element înainte ca elementul să fie afișat
filter: none | blur() | brightness() | contrast() | drop-shadow() | grayscale() | hue-rotate() | invert() | opacity() | saturate() | sepia() | url();
float Specifică dacă un element trebuie sau nu să fie pozitionat spre...
float: none|left|right|initial|inherit;
font Defineste proprietăți pentru stilul fontului, varianta font-ului, greutatea fontului, mărimea fontului / înălțimea liniei și proprietățile font-familie
font: italic bold 12px/30px Verdana, sans-serif;
@font-face Specifică regula care permite site-urilor web să descarce și să utilizeze alte fonturi decât fonturile "web-safe"
font-family: Lato; src: url(lato.woff);
font-style Specifică stilul fontului pentru text
font-style: normal|italic|oblique|initial|inherit;

G

  
grid Defineste sectiunea de subsol a paginii web
column-gap: 22px|normal|initial|inherit;
grid-area Fie specifică un nume pentru elementul grilă, fie această proprietate este o proprietate prescurtată pentru proprietățile grid-row-start, grid-column-start, grid-row-end și grid-column-end
grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end | itemname;
grid-auto-columns Specifică o dimensiune implicită a coloanei
.grid-container {display: grid; grid-auto-columns: 50px;}

H

  
height Setează înălțimea unui element
height: auto| 100%
hyphens Setează modul de împărțire a cuvintelor pentru a îmbunătăți aspectul paragrafelor
hyphens: none|manual|auto|initial|inherit;

I

  
image-rendering Oferă un indiciu browserului despre ce aspecte ale unei imagini sunt cele mai importante pentru a fi păstrate atunci când imaginea este scalată
image-rendering: auto| crisp-edges| pixelated;
@import Vă permite să importați o foaie de stil într-o altă foaie de stil
@import url|string list-of-mediaqueries;
isolation Definește dacă un element trebuie să creeze un nou conținut de stivuire
isolation: auto|isolate|initial|inherit;

J

  
justify-content Specifică alinierea dintre elementele din interiorul unui container flexibil atunci când elementele nu utilizează tot spațiul disponibil
justify-content: flex-start|flex-end|center|space-between|space-around|initial|inherit;

K

  
@keyframes Specifică codul css de animație
@keyframes numeanimatie {keyframes-selector {css-styles;}}

L

  
left Specifică poziția stângă a unui element
left: auto|length|initial|inherit;
letter-spacing Modifica spatiul dintre caractere dintr-un text
letter-spacing: normal|3px|initial|inherit;
line-height Setează înălțimea liniei (spatiul de deasupra si de desubtul unuei linii text)
line-height: normal|1.6|100%|initial|inherit;
list-style Setează toate proprietățile unei liste într-o singură declarație
list-style: list-style-type list-style-position list-style-image|initial|inherit;
list-style-position Specifică poziția marcatorilor pentru elementele listate (marcatori in sau inafara elementului parinte)
list-style-position: inside|outside|initial|inherit;
list-style-type Specifică tipul marcatorului de liste
list-style-type: circle|square|upper-roman|lower-alpha;

M

  
margin Setează toate proprietățile de margină într-o singură declarație
margin: 33px|auto|initial|inherit;
max-height Setează înălțimea maximă a unui element
max-height: none|50px|initial|inherit;
max-width Setează lățimea maximă a unui element
max-width: none|300px|initial|inherit;
@media Setează regulile de stil pentru diferite tipuri / dispozitive / dimensiuni media

min-height Setează înălțimea mminimă a unui element
min-height: none|50px|initial|inherit;
min-width Setează lățimea minimă a unui element
min-width: none|300px|initial|inherit;
mix-blend-mode Specifică modul în care conținutul unui element trebuie să se "amestece" cu elementul parinte
mix-blend-mode: normal|multiply|screen|overlay|darken|lighten|color-dodge|color-burn|difference|exclusion|hue|saturation|color|luminosity;

object-fit Specifică modul în care conținutul unui element înlocuit trebuie adaptat la elementul parinte stabilită de înălțimea și lățimea utilizată
object-fit: fill|contain|cover|scale-down|none|initial|inherit;
object-position Specifică alinierea elementului înlocuit în interiorul casetei
object-fit: left(right, center, 120px, 20%)|initial|inherit;
opacity Setează nivelul de opacitate pentru un element
opacity: 1(0,.5)|initial|inherit;
overflow Specifică ce se întâmplă dacă conținutul depășește caseta unui element
overflow: visible|hidden|scroll|auto|initial|inherit;

position: static|absolute|fixed|relative|sticky|initial|inherit;

white-space Specifică modul în care este manipulat spațiul alb din interiorul unui element

Selectori

În CSS, selectorii sunt modelele folosite pentru a selecta elementul (elementele) pe care doriți să le aplicati atribute CSS.

Selector Exemplu Descriere
.class .info Selecteaza toate elementele ce folosesc class="info"
.nume1.nume2 <div class="nume1 nume2">...</div> Selectează toate elementele cu nume1 și nume2 atribuind proprietatile css
#id <div id="caseta">...</div> Selectează elementul cu #caseta atribuind proprietatile css
* * Selectează toate elementele atribuind proprietatile css
element <p> Selectează toate elementele <p> atribuind proprietatile css
element, element <div>, <p> Selectează toate elementele <div> si <p> atribuind proprietatile css
element element <div> <p> Selectează toate elementele <p> din <div> atribuind proprietatile css
element > element <div> > <p> Selectează toate elementele <p> a caror element-parinte este <div> atribuind proprietatile css
element + element <div> + <p> Selectează toate elementele <p> daca sunt plasate imediat dupa elementul <div> atribuind proprietatile css
element1 ~ element2 <p> ~ <ul> Selectează toate elementele <ul> care sunt precedate de elementul <p> atribuind proprietatile css
[atribut] a[target] { background-color: green;} Selectează toate elementele ce contin atributul target (link-urile)
[atribut=valoare] [title~=produs] Selectează toate elementele cu atributul de titlu care conține cuvântul produs
[atribut^=valoare] a[href^="https"] Selectează fiecare element <a> a cărui valoare a atributului href începe cu https
[atribut$=valoare] a[href$=".pdf"] Selectează fiecare element <a> a cărui valoare a atributului href se termina cu extensia .pdf
[atribut*=valoare] a[href*="lenani"] Selectează toate elementele <a> a cărui valoare a atributului href contine segmentul lenani
:active a:active Selectează link-ul activ
::after p::after Inserați ceva după conținutul fiecărui element <p>. Exemplu: p::after{content:'+';margin-left: 5px;margin-right: 8px;}. Adauga semnul plus dupa elementul <p>
::before p::before Inserați ceva in fata conținutul fiecărui element <p>. Exemplu: p::before{content:'+';margin-left: 5px;margin-right: 8px;}. Adauga semnul plus in fata elementului <p>
:hover a:hover Aplică atributele CSS setate pentru acțiunea mouse deasupra unui link
:target #zenart:target Selectează elementul curent #zenart activ (clic pe o adresă URL care conține numele ancorei respective). Folosit cel mai des pentru navigarea in aceeasi pagina.

Nota: Unora dintre proprietatile CSS atribuite elementelor trebuiesc sa fie precedate de prefixul browserului pentru compatibilitate.
In acest context, -webkit-* / -moz-* / -o-* sunt prefixele browser-eleor cele mai utilizate

Exemplu:
img{ -webkit-border-radius: 6px;
border-radius: 6px; }
