CSS3 - Gradient / Pattern

Personal, pot afirma ca CSS este limitat doar de creativitatea web designer-ului.

CSS3 - Gradient

Gradienții CSS vă permit să afișați tranziții ușoare între două sau mai multe culori specificate.

Pentru a crea un gradient liniar, trebuie să definiți cel puțin două opriri de culoare. Opririle de culori sunt culorile pe care doriți să le faceți între tranziții ușoare. De asemenea, puteți seta un punct de pornire și o direcție (sau un unghi) împreună cu efectul de gradient.

background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

CSS3 - Pattern

Consultați următoarele modele de fundal CSS pe care le puteți adăuga pe site-ul dvs. web. CSS3 vă permite să creați câteva modele de fundal complexe. Acestea sunt doar câteva.

Modelele CSS (css patterns) folosesc un concept similar cu Gradienturile CSS, diferența fiind vizibilă in codul css iar culorile se opresc brusc si nu in trepte la fel ca Gradientul.
